# DLT
A Blockchain Decentralized Application For Fighting Counterfeited Documents And Products Such As Land Titles, Food Products, Cosmetics , Pharmaceuticals, Academic Documents &amp; More 

## Introduction
The system is based of ERC-1155 standard. The contracts are written based off OpenZeppelin.

## Architecture

- OpenZeppelin Contracts v5
- Hardhat
- Ethers.js
- Node.js
